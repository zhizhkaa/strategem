Справка по конфигурации

Файлы используют YAML. Отступы задают структуру, табы не применяются.

parameters.yaml описывает параметры, значения по умолчанию, границы и признак ввода.

formulas.yaml содержит формулы. Доступны функции min, max, round, prev, mean, interpolate, max_loan.

decision_order.yaml задаёт этапы решений, министров, порядок расчётов и режимы прохождения.

interpolation.yaml хранит таблицы с одинаковой длиной списков x и y.

Перед сохранением запускается полная проверка всех конфигурационных файлов.

Сбросить файл?
